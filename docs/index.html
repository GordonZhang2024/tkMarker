tkMarker¶

A Markdown editor using Tkinter

  • Markdown preview

  • Basic markdown syntax support

Installation¶

Installing the dependencies¶

This project requires Pyinstaller and Tkinter.

GNU/Linux¶

To install them, type command:

$ sudo dnf install python3-tkinter #On Fedora, CentOS, and Red Hat
$ pip install pyinstaller

Installing tkMarker¶

GNU/Linux¶

Then, type command:

$ git clone 'https://github.com/GordonZhang2024/tkMarker.git'
$ cd tkMarker
$ ./build.sh
$ sudo ./install.sh

IMPORTANT: Please use Python >= 3.6

Usage¶

Type command:

$ tkmarker

You will see the editor.

Click the Preview Button, then the editor will open the preview in the web browser.
